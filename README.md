# tg-chat-bot

Telegram бот для групповых бесед с анкетами, браками и весёлыми командами.

> **Примечание:** Telegram поддерживает в меню бота только латинские команды (`[a-z0-9_]`). Кириллические варианты работают, но не отображаются в меню — используй любой вариант.

## Команды

| Команда | Алиас | Описание |
|---|---|---|
| `/info` (reply) | — | Показать анкету пользователя |
| `/info @username` | — | Показать анкету по юзернейму |
| `/anketa` | `/анкета` | Заполнить/обновить свою анкету |
| `/brak` (reply / @username) | — | Сделать предложение брака |
| `/razvod` (reply / @username) | `/развод` | Расторгнуть брак |
| `/families` | — | Все семьи и как долго они в браке |
| `/orgy` | `/оргия` | Запустить опрос (1 раз в сутки) |

## Как работает брак

- Пользователь пишет `/brak` в ответ на сообщение человека **или** `/brak @username`.
- Бот отправляет предложение брака с кнопками **Да / Нет**.
- Нажать кнопку может только тот человек, которому сделали предложение.
- Если он нажимает **Да**, брак сохраняется в базе с датой создания.
- Если **Нет**, предложение удаляется.
- В `/families` бот показывает, кто с кем в семье и сколько времени семья уже существует.

## Сообщения при обновлении

Бот запоминает чаты, где его использовали, и:
- при запуске пишет: `бля, я ожил и обновился`
- при корректной остановке пишет: `бля, умираю`

> Если процесс убьют жёстко (`kill -9`, crash, падение контейнера без graceful shutdown), сообщение про выключение может не успеть отправиться.

## Быстрый старт

### 1. Создай бота
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`, задай имя и юзернейм
3. Скопируй токен

### 2. Включи необходимые настройки бота
В @BotFather:
- `/setprivacy` → выбери своего бота → `Disable`  
  _(чтобы бот читал все сообщения в группе)_

### 3. Клонируй репозиторий
```bash
git clone https://github.com/SkullofGods/tg-chat-bot.git
cd tg-chat-bot
```

### 4. Настрой окружение
```bash
cp .env.example .env
# Открой .env и вставь свой BOT_TOKEN
```

### 5. Установи зависимости
```bash
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 6. Запусти бота
```bash
python bot.py
```

### 7. Добавь бота в беседу
1. Добавь бота в групповой чат
2. Сделай его **администратором** (нужны права: читать сообщения, удалять сообщения, управлять опросами)
3. Группа должна быть **супергруппой** (если нет — промотируй любым действием в настройках)

> **Важно:** Для получения событий о новых участниках группа должна быть супергруппой.

## Запуск через Docker (опционально)

```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
CMD ["python", "bot.py"]
```

```bash
docker build -t tg-chat-bot .
docker run -d --env-file .env --name tg-chat-bot tg-chat-bot
```
